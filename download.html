The Fun Doesn't Stop Here

Our smart contract design allows for the deck of cards to be used in making other playing card games. Many games outside of cards would be a great fit for this private table model as well. Although we are calling them private tables it is possible to use them in sit & go, or tournament style applications as we've demonstrated with our five card tables. Any currencies in the Dero network can be added to the tables. For customization, NFT integration would make each table unique. Stay tuned for updates. If you'd like a custom table built feel free to contact us about your inquiry.
